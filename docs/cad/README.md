# CAD

This section is reserved for final CAD files and enclosure drawings.

## Current Status

CAD files are to be added. The current project has focused on functional prototyping, wiring, software integration, and documentation.

## Planned Enclosure Requirements

- Hold the Raspberry Pi securely.
- Provide openings for USB power, HDMI/service access, and audio devices.
- Mount the OLED where it is visible to the user.
- Place the microphone and voice sensor where speech can be detected clearly.
- Include ventilation for Raspberry Pi heat.
- Keep GPIO and relay wiring protected.
